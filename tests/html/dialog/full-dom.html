Dialog Box :: Full DOM test

In this test, the DOM for the dialog box is fully defined within the HTML document, the script just takeover to manage its state. The dialog box provide two buttons to close it. The first button is using the instance Dialog::close() method, when the second button use the static Dialog.close() method.
